# Q4 2025 Earnings Report Presentation

An interactive RevealJS presentation showcasing quarterly financial performance for stakeholders.

## Features

- **Markdown Support**: Executive summary written in Markdown
- **Animated Elements**: Fragment animations for progressive disclosure
- **Code Highlighting**: Python, JavaScript, and SQL code samples with syntax highlighting
- **Mathematical Formulas**: Financial equations rendered using KaTeX
- **Speaker Notes**: Comprehensive presentation guidance for each slide
- **Responsive Design**: Works on desktop and mobile devices

## Presentation Contents

1. **Title Slide**: Company branding and contact information
2. **Executive Summary** (Markdown): Key highlights and strategic initiatives
3. **Financial Metrics**: Animated table showing YoY performance
4. **Revenue Model**: Python code for revenue calculations
5. **Financial Ratios**: ROE, EPS, and P/E ratio formulas
6. **Portfolio Analysis**: Modern Portfolio Theory equations
7. **Risk Metrics**: JavaScript implementation of VaR calculation
8. **Growth Drivers**: Animated visual journey through quarters
9. **Strategic Outlook**: Highlighted future initiatives
10. **Data Analytics**: SQL query for segment analysis
11. **Q&A Slide**: Contact information and keyboard shortcuts

## Keyboard Shortcuts

- **Arrow Keys / Space**: Navigate slides
- **F**: Enter fullscreen mode
- **S**: Open speaker notes window
- **O**: Toggle overview mode
- **ESC**: Exit overview/fullscreen
- **?**: Show help overlay

## Technologies Used

- **RevealJS 4.5.0**: HTML presentation framework
- **KaTeX**: Math equation rendering
- **Highlight.js**: Code syntax highlighting
- **Markdown Plugin**: Markdown slide support

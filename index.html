Portfolio Risk Analysis

Using Modern Portfolio Theory to optimize our investment allocation:

\[\mu_p = \sum_{i=1}^{n} w_i \mu_i\]

Expected portfolio return

\[\sigma_p^2 = \sum_{i=1}^{n}\sum_{j=1}^{n} w_i w_j \sigma_i \sigma_j \rho_{ij}\]

Portfolio variance with correlation

\[\text{Sharpe Ratio} = \frac{\mu_p - r_f}{\sigma_p} = \frac{0.124 - 0.045}{0.082} = 0.96\]

Risk-adjusted return measure

Our portfolio optimization uses Modern Portfolio Theory. Expected return is weighted average of individual asset returns. Portfolio variance accounts for correlations between assets. Sharpe Ratio of 0.96 indicates good risk-adjusted performance.

Risk Metrics Implementation


// Calculate Value at Risk (VaR)
const calculateVaR = (returns, confidence = 0.95) => {

  // Sort returns in ascending order
  const sortedReturns = returns.sort((a, b) => a - b);

  // Find the percentile index
  const index = Math.floor((1 - confidence) * returns.length);
  const var95 = sortedReturns[index];

  // Calculate Conditional VaR (Expected Shortfall)
  const cvar = sortedReturns
    .slice(0, index)
    .reduce((sum, val) => sum + val, 0) / index;

  return { var95, cvar };
};

// Example: Portfolio of $100M
// VaR(95%) = $4.2M maximum expected loss
This JavaScript implementation calculates Value at Risk. VaR estimates the maximum expected loss at a given confidence level. CVaR (Conditional VaR) shows the expected loss beyond VaR threshold. Our current portfolio VaR(95%) is $4.2M on $100M portfolio.

Data Analytics Query


-- Quarterly revenue analysis by product segment
SELECT
    product_segment,
    SUM(revenue) as total_revenue,
    AVG(profit_margin) as avg_margin,
    COUNT(DISTINCT customer_id) as customer_count,
    SUM(revenue) / SUM(SUM(revenue)) OVER () * 100 as revenue_share
FROM sales_data
WHERE quarter = 'Q4'
  AND year = 2025
GROUP BY product_segment
ORDER BY total_revenue DESC;
This SQL query analyzes revenue distribution across product segments. We aggregate revenue, profit margins, and customer counts. The window function calculates each segment's revenue share. This analysis helps identify high-performing segments for investment.
